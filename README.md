🎓 Student Help Desk Application (ServiceNow)

📌 Project Overview

The **Student Help Desk Application** is a custom-built ServiceNow solution designed to streamline the process of managing student service requests. This application automates request handling and improves response efficiency using ServiceNow Flow Designer.

🚀 Key Features
 🗂️ Custom Table Creation
* Created a `Student Request` table to store request details.

📝 Form Design
 * Designed user-friendly form with fields:
    * Student Name
    * Student ID
    * Issue Type
    * Description
    * Priority
    * Status
⚙️ Workflow Automation (Flow Designer)
  * Trigger: Record creation in Student Request table
  * Automatically processes requests using flow logic

📧 Email Notification
Sends automated email when a new request is created

🛠️ Technologies Used

* ServiceNow Platform
* Flow Designer
* Email Notification System

How It Works
1. User submits a new Student Request
2. Flow Designer triggers on record creation
3. System processes the request automatically
4. Email notification is sent to the configured recipient
